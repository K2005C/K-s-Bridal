# K's Bridal

A responsive bridal boutique website built using HTML, CSS and JavaScript.

## Live Demo

[View Live Demo](https://k2005c.github.io/K-s-Bridal/)

## Features

- Responsive design for desktop and mobile devices
- Bridal dress category filtering
- Interactive appointment booking form
- Custom success popup using JavaScript
- Modern bridal-themed user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Design
- CSS Flexbox
- CSS Grid
- DOM Manipulation
